## 4. DERIVATIONS

### D1. SUVAT Equations from Constant Acceleration
**Starting point:** a = dv/dt = constant [MUST MEMORISE]
1. Integrate a = dv/dt: v = u + at
2. Integrate v = dx/dt: s = ut + (1/2)at^2
3. Eliminate t between (1) and (2): v^2 = u^2 + 2as
4. Average velocity: s = (1/2)(u+v)t

### D2. Work-Energy Theorem
**Starting point:** F = ma [MUST MEMORISE]
1. W = integral F dx = integral m(dv/dt) dx
2. Use dx = v dt: W = integral mv dv
3. W = (1/2)mv_f^2 - (1/2)mv_i^2 = Delta KE

### D3. Conservation of Energy
**Starting point:** F = -dU/dx [ON SHEET], F = ma [MUST MEMORISE]
1. dE/dt = d(KE)/dt + d(PE)/dt
2. = mv(dv/dt) + (dU/dx)(dx/dt) = v(ma) + v(dU/dx)
3. = v(F + dU/dx) = v(F - F) = 0

### D4. Escape Velocity
**Starting point:** Conservation of energy, U = -GMm/r [ON SHEET]
1. At surface: (1/2)mv_e^2 - GMm/R = 0
2. v_e = sqrt(2GM/R)

### D5. Tsiolkovsky Rocket Equation
**Starting point:** Conservation of momentum
1. At time t: momentum = mv
2. At t+dt: (m+dm)(v+dv) + (-dm)(v-u) = mv
3. Simplify: m dv = -u dm
4. Integrate: Delta v = u ln(m_0/m_f)

### D6. Centripetal Acceleration
**Starting point:** r(t) = r cos(omega t) i-hat + r sin(omega t) j-hat
1. v = dr/dt = -r omega sin(omega t) i-hat + r omega cos(omega t) j-hat
2. a = dv/dt = -r omega^2 cos(omega t) i-hat - r omega^2 sin(omega t) j-hat = -omega^2 r
3. |a| = omega^2 r = v^2/r

### D7. Atwood Machine
**Starting point:** Newton's second law for each mass
1. m_1: m_1 g - T = m_1 a
2. m_2: T - m_2 g = m_2 a
3. Add: a = (m_1 - m_2)g/(m_1 + m_2)
4. Substitute back: T = 2m_1 m_2 g/(m_1 + m_2)

### D8. Reduced Mass
**Starting point:** Newton's third law: F_12 = -F_21
1. m_1 a_1 = F_12, m_2 a_2 = -F_12
2. a_rel = a_1 - a_2 = F_12(1/m_1 + 1/m_2)
3. Define 1/mu = 1/m_1 + 1/m_2, so mu a_rel = F_12

### D9. SHM Solution from Equation of Motion
**Starting point:** d^2x/dt^2 + omega_0^2 x = 0
1. Trial solution: x = A cos(omega_0 t + phi)
2. d^2x/dt^2 = -A omega_0^2 cos(omega_0 t + phi) = -omega_0^2 x. Confirmed.

### D10. Damped SHM Solution
**Starting point:** d^2x/dt^2 + gamma dx/dt + omega_0^2 x = 0
1. Trial: x = C e^(pt), p = alpha + i omega_d
2. Substitute: p^2 + gamma p + omega_0^2 = 0
3. Imaginary part: alpha = -gamma/2
4. Real part: omega_d = sqrt(omega_0^2 - gamma^2/4)
5. Solution: x(t) = A e^(-gamma t/2) cos(omega_d t + phi)

### D11. Forced Oscillation Steady-State
**Starting point:** d^2x/dt^2 + gamma dx/dt + omega_0^2 x = (F_0/m)cos(omega t)
1. Trial: x_ss = A cos(omega t - delta)
2. Substitute and match coefficients
3. A(omega) = (F_0/m)/sqrt[(omega_0^2 - omega^2)^2 + gamma^2 omega^2]
4. tan(delta) = gamma omega/(omega_0^2 - omega^2)

### D12. Wave Equation for a String
**Starting point:** Newton's second law on string element, small angle approximation
1. Net force on segment: F_y = T(dy/dx|_{x+dx} - dy/dx|_x) = T(d^2y/dx^2)dx
2. Mass of segment: dm = mu dx
3. F = ma: T(d^2y/dx^2)dx = mu dx (d^2y/dt^2)
4. d^2y/dx^2 = (mu/T) d^2y/dt^2, with v^2 = T/mu

### D13. Reflection and Transmission Coefficients
**Starting point:** Boundary conditions at junction
1. Continuity of displacement: A_i + A_r = A_t
2. Continuity of force: Z_1(A_i - A_r) = Z_2 A_t (from T dy/dx continuity)
3. Solve: r = (Z_1 - Z_2)/(Z_1 + Z_2), t = 2Z_1/(Z_1 + Z_2)

### D14. Standing Wave from Superposition
**Starting point:** Two counter-propagating waves
1. y_1 = A sin(kx - omega t), y_2 = A sin(kx + omega t)
2. y = y_1 + y_2 = 2A sin(kx) cos(omega t) [using trig identity]

### D15. Coupled Oscillator Normal Modes
**Starting point:** Equations of motion for two coupled masses
1. m d^2x_A/dt^2 = -kx_A - k_c(x_A - x_B)
2. m d^2x_B/dt^2 = -kx_B - k_c(x_B - x_A)
3. Define q_1 = x_A + x_B, q_2 = x_A - x_B
4. d^2q_1/dt^2 = -omega_0^2 q_1 (omega_1 = omega_0)
5. d^2q_2/dt^2 = -(omega_0^2 + 2omega_c^2) q_2 (omega_2 = sqrt(omega_0^2 + 2k_c/m))

### D16. Hermitian Eigenvalues are Real
**Starting point:** A|a> = a|a>, A^dagger = A
1. <a|A|a> = a<a|a>
2. Take conjugate: <a|A^dagger|a> = a* <a|a>
3. Since A = A^dagger: a = a*

### D17. Orthogonality of Hermitian Eigenvectors
**Starting point:** A|a> = a|a>, A|b> = b|b>, a != b
1. <b|A|a> = a<b|a>
2. <a|A|b>* = b<b|a> (since b real)
3. Since A = A^dagger: (a-b)<b|a> = 0
4. a != b implies <b|a> = 0

### D18. Time Evolution from Schrodinger Equation
**Starting point:** i hbar d|psi>/dt = H|psi> [ON SHEET]
1. Expand: |psi(t)> = Sum c_n(t)|E_n>
2. i hbar dc_n/dt = E_n c_n(t) for each n
3. Solution: c_n(t) = c_n(0) e^(-iE_n t/hbar)

### D19. Adiabatic PV^gamma = const
**Starting point:** First law with Q=0: dU = -PdV, dU = nC_V dT, PV = nRT [ON SHEET]
1. nC_V dT = -PdV = -(nRT/V)dV
2. C_V dT/T = -R dV/V
3. Integrate: C_V ln(T) = -R ln(V) + const
4. T V^(R/C_V) = const, and R/C_V = gamma - 1
5. Using PV = nRT: PV^gamma = const

### D20. Carnot Efficiency
**Starting point:** Isothermal + adiabatic processes, PV = nRT [ON SHEET]
1. Isothermal at T_h: Q_h = nRT_h ln(V_2/V_1)
2. Isothermal at T_c: Q_c = nRT_c ln(V_3/V_4)
3. Adiabatic relations give V_2/V_1 = V_3/V_4
4. eta = 1 - Q_c/Q_h = 1 - T_c/T_h

## 5. PROBLEM ARCHETYPES

### A. Mechanics Problem Types

**[EASY] A-E1: Dimensional Analysis**
- Given a physical situation, determine the plausible equation using dimensional analysis
- Check if a given equation is dimensionally consistent
- Typical: "Use dimensional analysis to find the dependence of period on length, mass, g for a pendulum"

**[EASY] A-E2: SUVAT Application**
- Direct application of one or two SUVAT equations
- Given 3 of 5 variables, find the others
- Typical: "A ball is thrown upward at 20 m/s. Find max height, time to return."

**[MEDIUM] A-M1: Projectile Motion**
- 2D SUVAT with horizontal and vertical components
- Find range, max height, time of flight, velocity at impact
- Typical: "A projectile is launched at 30 m/s at 45 degrees. Find range and max height."
- Values: speeds 10-50 m/s, angles 30-60 degrees, g = 9.8 m/s^2

**[MEDIUM] A-M2: Newton's Second Law with Friction**
- Free body diagram, resolve forces, apply F = ma
- Inclined planes with friction, connected masses
- Typical: "A 5 kg block on a 30-degree incline with mu_k = 0.3. Find acceleration."
- Values: masses 2-20 kg, angles 20-60 degrees, mu = 0.1-0.8

**[MEDIUM] A-M3: Work-Energy Problems**
- Apply work-energy theorem or conservation of energy
- Springs, gravity, friction as energy sources/sinks
- Typical: "A 2 kg mass slides from rest down a 3 m frictionless ramp (30 degrees). Find speed at bottom."

**[MEDIUM] A-M4: Momentum and Collisions**
- Apply conservation of momentum, classify collision type
- 1D elastic/inelastic collisions, impulse calculations
- Typical: "A 3 kg ball moving at 4 m/s collides elastically with a 1 kg ball at rest. Find final velocities."

**[MEDIUM] A-M5: Circular Motion**
- Find centripetal force/acceleration, apply to physical systems
- Banked curves, vertical loops, orbits
- Typical: "Find the minimum speed at the top of a vertical loop of radius 5 m."

**[HARD] A-H1: Rotational Dynamics**
- Moment of inertia calculations, torque, angular acceleration
- Rolling objects, Atwood machine with pulley, angular momentum conservation
- Typical: "A solid sphere rolls without slipping down a 30-degree incline. Find its acceleration."
- Requires: I_sphere = (2/5)MR^2, rolling constraint v = R omega

**[HARD] A-H2: Multi-Concept Problems**
- Combine energy conservation with rotational mechanics
- Include both translational and rotational KE
- Typical: "Compare the speeds at the bottom of a ramp for a disk, sphere, and ring of equal mass and radius."

**[HARD] A-H3: Non-Uniform Acceleration**
- Integrate equations of motion for position/velocity-dependent forces
- Linear drag problems (terminal velocity, exponential approach)
- Typical: "An object falls with drag F = -bv. Find v(t) and terminal velocity."

**[EXTREME] A-X1: Complex Rotational + Friction Problems**
- Snooker ball struck off-centre: find height for pure rolling
- Compound pendulum problems
- Combined angular momentum conservation with energy methods
- Typical: "A snooker ball of radius r is struck at height h. At what h does it immediately roll without slipping?"

**[EXTREME] A-X2: Two-Body Reduced Mass Problems**
- Transform to reduced mass, solve, transform back
- Two beads on a spring, orbital problems
- Typical: "Two masses m_1 and m_2 connected by spring k. Find oscillation frequency."

### B. Properties of Matter Problem Types

**[EASY] B-E1: Heat Capacity Calculations**
- Direct Q = mc Delta T or Q = mL
- Thermal equilibrium of two bodies
- Typical: "How much energy to heat 2 kg of water from 20 C to 100 C?"
- Values: c_water = 4200 J/(kg K), typical masses 0.5-5 kg

**[EASY] B-E2: Ideal Gas Law**
- Direct PV = nRT
- Find one variable given others
- Typical: "What volume does 1 mol of ideal gas occupy at STP?"

**[MEDIUM] B-M1: Thermodynamic Processes**
- Calculate W, Q, Delta U for isothermal/isobaric/isochoric/adiabatic processes
- Sketch PV diagrams
- Typical: "2 mol of monatomic gas expand isothermally at 300 K from 10 L to 20 L. Find W, Q, Delta U."

**[MEDIUM] B-M2: Adiabatic Processes**
- Apply PV^gamma = const, TV^(gamma-1) = const
- Find final T, P, or V after adiabatic compression/expansion
- Typical: "A diesel engine compresses air (gamma=1.4) by factor 15. If T_initial = 300 K, find T_final."

**[MEDIUM] B-M3: Entropy Calculations**
- Delta S = mc ln(T2/T1), Delta S = Q/T
- Total entropy change of system + reservoir
- Typical: "1 kg of water at 80 C is placed in contact with reservoir at 20 C. Find Delta S_total."

**[HARD] B-H1: Complete Thermodynamic Cycles**
- Analyse multi-step cycles on PV diagram
- Calculate W, Q, Delta U for each step and the full cycle
- Efficiency of the cycle
- Typical: "A cycle consists of: isothermal expansion at T_h, isochoric cooling, isothermal compression at T_c, isochoric heating. Find efficiency."

**[HARD] B-H2: Carnot Engine Problems**
- Carnot efficiency, finite reservoir problems
- Engine-refrigerator combinations
- Typical: "A Carnot engine operates between 500 K and 300 K, absorbing 1000 J per cycle. Find W and Q_c."

**[HARD] B-H3: Lennard-Jones Potential**
- Find equilibrium separation, binding energy
- Compare binding energy to thermal energy to predict phase
- Typical: "Given LJ potential with epsilon and a, find equilibrium separation and determine if substance is gas or solid at room T."

**[EXTREME] B-X1: Multi-Reservoir Entropy Problems**
- Approaching reversibility with intermediate baths
- Free expansion entropy with non-ideal processes
- Typical: "Show that Delta S_total -> 0 as the number of intermediate reservoirs -> infinity."

**[EXTREME] B-X2: Maxwell-Boltzmann Distribution**
- Derive most probable / mean / rms speed
- Atmospheric escape calculations
- Typical: "Derive the most probable speed from the MB distribution. Can helium escape Earth's atmosphere?"

### C. Oscillations and Waves Problem Types

**[EASY] C-E1: SHM Parameters**
- Extract amplitude, frequency, period, phase from x(t) = A cos(omega t + phi)
- Typical MCQ: "A particle oscillates as x = 3 cos(4t + pi/3). What is the period?"

**[EASY] C-E2: SHM Energy**
- Total energy E = (1/2)kA^2
- Find max velocity from energy
- Typical: "A 0.5 kg mass on spring (k=200 N/m) oscillates with A=0.1 m. Find max speed."

**[EASY] C-E3: Standing Wave Identification**
- Identify nodes and antinodes
- Count harmonics
- Typical MCQ: "A string of length 2 m vibrates in 3rd harmonic. What is the wavelength?"

**[MEDIUM] C-M1: Damped Oscillation Analysis**
- Classify damping regime, find damped frequency
- Calculate energy decay rate
- Typical: "Given m=0.5 kg, k=50 N/m, b=2 kg/s. Find omega_d and Q factor."

**[MEDIUM] C-M2: Forced Oscillation Resonance**
- Calculate steady-state amplitude at given driving frequency
- Find resonance conditions and peak amplitude
- Typical: "An oscillator with omega_0=10 rad/s, Q=20 is driven at omega=9 rad/s. Find amplitude ratio."

**[MEDIUM] C-M3: Wave Speed and Properties**
- Calculate wave speed from medium properties
- Find wavelength/frequency relationships
- Typical: "A string of mu=0.01 kg/m under T=100 N. Find wave speed and wavelength for f=500 Hz."

**[MEDIUM] C-M4: Impedance and Reflection**
- Calculate reflection/transmission coefficients from impedances
- Power reflection and transmission
- Typical: "Two strings with mu_1=0.01, mu_2=0.04 kg/m joined under same tension. Find r, R."

**[MEDIUM] C-M5: Doppler Effect**
- Apply Doppler formula for moving source/observer
- Typical MCQ: "An ambulance (f=500 Hz) approaches at 30 m/s. What frequency is heard? (v_sound=340 m/s)"

**[HARD] C-H1: Coupled Oscillators**
- Find normal mode frequencies and mode shapes
- Determine motion from initial conditions
- Energy transfer / beats
- Typical: "Two identical masses (m) connected by springs (k=2 N/m, k_c=0.5 N/m). Find normal mode frequencies."

**[HARD] C-H2: Wave Energy and Power Derivation**
- Derive energy per unit length and power transmitted
- Typical: "Show that <P> = (1/2) mu omega^2 A^2 v for a harmonic wave on a string."

**[HARD] C-H3: Complex Superposition Problems**
- Add multiple waves using phasors
- Beats and group velocity
- Typical: "Three waves of equal amplitude but phases 0, 2pi/3, 4pi/3 superpose. Find resultant amplitude."

**[EXTREME] C-X1: Dispersion and Group Velocity**
- Derive group velocity from dispersion relation
- Compare v_phase and v_group
- Typical: "Given omega(k) = sqrt(omega_0^2 + c^2 k^2), find v_phase and v_group."

**[EXTREME] C-X2: Physical Pendulum + Parallel Axis**
- Combine rotational mechanics with oscillation theory
- Typical: "A uniform rod of length L pivoted at distance L/4 from one end. Find period of small oscillations."

### D. Quantum Physics Problem Types

**[EASY] D-E1: Normalisation**
- Check if state is normalised; find normalisation constant
- Typical MCQ: "Is |psi> = (1/sqrt(3))|1> + sqrt(2/3)|2> normalised?"

**[EASY] D-E2: Measurement Probabilities (Direct)**
- Given state in eigenbasis, find probability of each outcome
- Typical MCQ: "For |psi> = (3/5)|E_1> + (4/5)|E_2>, what is P(E_1)?"

**[EASY] D-E3: Identify Dirac Notation Elements**
- Classify as ket, bra, operator, scalar, or nonsense
- Typical MCQ: "Which of <a|b>, |a><b|, <a||b>, |a>|b> is an operator?"

**[EASY] D-E4: Particle Physics Facts**
- Quark content of proton/neutron, conservation laws
- Typical MCQ: "What is the charge of a proton in terms of quark charges?"

**[MEDIUM] D-M1: Inner Products and Orthogonality**
- Compute <phi|psi> for given states
- Verify orthogonality
- Typical: "Show that |up_x> and |down_x> are orthogonal."

**[MEDIUM] D-M2: Operator Properties**
- Test if operator is Hermitian or unitary
- Find Hermitian conjugate of given operator
- Typical: "Is A = |1><2| + |2><1| Hermitian? Unitary?"

**[MEDIUM] D-M3: Eigenvalue Problems**
- Find eigenvalues and eigenvectors of 2x2 operators
- Typical: "Find eigenvalues of A = E_0|L><L| - E_0|R><R|."

**[MEDIUM] D-M4: Spin Measurement Sequences**
- Sequential measurements in different bases
- Typical: "Particle starts in |up_z>. S_x is measured, then S_z. Find probabilities."

**[HARD] D-H1: Time Evolution**
- Expand state in energy eigenbasis, apply time evolution
- Calculate probability oscillations
- Typical: "Given H with eigenstates |+> and |-> and energies E_1, E_2. Initial state |L>. Find P(L,t)."

**[HARD] D-H2: Mach-Zehnder Interferometer**
- Apply beam-splitter, mirror, phase-shifter operators in sequence
- Calculate detection probabilities
- Typical: "A MZ interferometer has phase shifter theta = pi/4 in top arm. Find P(detector A)."

**[HARD] D-H3: Entanglement and Multi-Particle States**
- Determine if two-particle state is entangled
- Calculate measurement probabilities for composite systems
- Typical: "Is (1/sqrt(2))(|up>|down> + |down>|up>) entangled? What is P(both up) for S_z measurement?"

**[EXTREME] D-X1: Combined Time Evolution + Measurement**
- Time evolve, then measure in non-energy basis
- Typical: "State starts in |L>, evolves under H for time t. Measure position. Find P(L,t) and P(R,t)."

**[EXTREME] D-X2: Identical Particles**
- Construct properly (anti)symmetrised states
- Verify Pauli exclusion
- Typical: "Two identical fermions in a 3-state system. Write the most general antisymmetric state."

**[EXTREME] D-X3: Particle Physics Analysis**
- Analyse allowed/forbidden decays using conservation laws
- Identify forces responsible for given processes
- Typical: "Is the decay pi+ -> mu+ + nu_mu allowed? What force mediates it? Check all conservation laws."
